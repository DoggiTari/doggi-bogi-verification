2. Descripción del Servicio

DoggiBogi es un bot de Discord multifuncional que proporciona las siguientes funcionalidades:

  • Sistema de Moderación Avanzado: Herramientas de moderación incluyendo sistema de advertencias, expulsiones, baneos, softbans, gestión de roles, logs de moderación y sistema de inmunidad.
  • Sistema de Perfiles Personalizados: Perfiles únicos con sistema de XP automático, niveles dinámicos, medallas personalizables, biografías personalizadas, fondos exclusivos para boosters y sistema de ranking.
  • Herramientas de Utilidades: Comandos útiles como obtención de IDs de emojis, información del servidor, información de usuarios, generación de colores aleatorios y sistema de invitación del bot.
  • Sistema de Administración: Herramientas de configuración avanzadas para administradores, gestión de prefijos, configuración de canales de XP, sistema de logs de nivel y herramientas de gestión del servidor.
  • Sistema Snipe: Recuperación temporal de mensajes borrados con almacenamiento seguro y permisos especiales para usuarios con privilegios.
  • Sistema de Entretenimiento: Comandos de diversión, sistema de inventario, códigos de recompensa, medallas automáticas y actividades interactivas para la comunidad.
  • Analytics y Estadísticas: Métricas detalladas del servidor, logs de actividad, sistema de ranking y estadísticas de uso en tiempo real.

El bot está diseñado para mejorar la experiencia de los usuarios en Discord proporcionando herramientas útiles y entretenimiento para las comunidades.

4. Privacidad y Protección de Datos

Nos comprometemos a proteger su privacidad y manejar sus datos de manera responsable. El bot puede recopilar y almacenar los siguientes tipos de datos:

  • Datos de Usuario: ID de Discord, nombre de usuario, avatar, información de perfil y preferencias de usuario.
  • Datos de Servidor: ID del servidor, configuraciones, roles, canales y configuraciones de moderación.
  • Datos de Actividad: Comandos utilizados, frecuencia de uso, estadísticas de participación y métricas de interacción.
  • Datos de Moderación: Historial de advertencias, acciones de moderación, logs de actividad y registros de infracciones.
  • Datos de Perfil: XP, niveles, medallas, biografías personalizadas, configuraciones de perfil y preferencias de visualización.
  • Mensajes Temporales: Contenido de mensajes borrados (para funcionalidad snipe) que se almacenan temporalmente y se eliminan automáticamente.

Uso de Datos: Los datos recopilados se utilizan exclusivamente para:

  • Proporcionar las funcionalidades del bot
  • Mantener perfiles de usuario y sistemas de progreso
  • Gestionar sistemas de moderación y seguridad
  • Mejorar el rendimiento y funcionalidad del bot
  • Generar estadísticas y analytics del servidor
🔒 Compromiso de Privacidad: No vendemos, alquilamos ni compartimos sus datos personales con terceros. Todos los datos se almacenan de forma segura y se utilizan únicamente para el funcionamiento del bot.
